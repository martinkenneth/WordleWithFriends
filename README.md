# Wordle With Friends

Full stack web app with the purpose of creating and sharing your own *[Wordle](https://www.nytimes.com/games/wordle/index.html)* with friends, using: React, Java, Spring Boot, and MySQL

## Technologies:

### Frontend
- HTML
- CSS
- React
  - Crypto-JS
  - React-Router-Dom
  - Axios

### Backend
- Java
- Spring Boot

### Database
- MySQL
